# Spam Email Detection using Naïve Bayes
This is a Machine Learning mini-project that classifies emails as **Spam** or **Ham**.

## Features
- **Algorithm:** Multinomial Naïve Bayes
- **UI:** Interactive Tkinter Dashboard
- **Dataset:** SMS Spam Collection

## How to Run
1. Install dependencies: `pip install pandas scikit-learn nltk`
2. Run the app: `python app.py`
